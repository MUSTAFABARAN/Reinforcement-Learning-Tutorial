# Reinforcement-Learning-Tutorial
Sample reinforcement learning notebooks

## Environment and tools

1. Jupyter Notebook
2. Numpy
3. gym
4. pytorch
5. Matplotlib

## To Execute

`pip install numpy`

`pip install gym`

`pip install torch`

`pip install matplotlib`

`jupyter notebook`
